## 使用方法

- 部署
    - gitee `hexo clean && hexo generate --config _config-gitee.yml && hexo d --config _config-gitee.yml`
        - 部署静态站点
    - github `git push`
        - ci自动编译打包，发布到gh-pages分支
        - github 自动部署gh-pages到gitpage服务
